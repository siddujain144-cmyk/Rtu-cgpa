# RTU SGPA & CGPA Calculator - Quick Start Guide ⚡

## Installation & Setup

### Step 1: Open the Application
Simply open `index.html` in your web browser. No installation needed!

---

## 3 Quick Ways to Calculate Your GPA

### 🔵 Way 1: Manual Entry (Fastest)
**Time: 2 minutes per semester**

```
1. Select Branch → Select Semester
2. Enter each subject's grade from dropdown
3. Click "Calculate SGPA"
4. Click "Save SGPA"
✅ Done!
```

### 🔵 Way 2: Upload Marksheet (Automatic)
**Time: 1 minute per semester**

```
1. Go to "Upload Marksheet" tab
2. Drag & drop your marksheet photo
3. System auto-detects grades
4. Select Semester
5. Click "Confirm & Calculate"
✅ Done!
```

### 🔵 Way 3: View Final Results
**Time: Instant**

```
1. Go to "View Results" tab
2. See all your SGPAs
3. View automatic CGPA
✅ Perfect!
```

## Key Information 📚

### Grading Scale
| Grade | Points |
|-------|--------|
| A+/A  | 4.0    |
| B+    | 3.5    |
| B     | 3.0    |
| C+    | 2.5    |
| C     | 2.0    |
| D     | 1.0    |
| F     | 0.0    |

### Formula
```
SGPA = Sum(Grade × Credits) / Sum(Credits)
CGPA = Sum(All SGPAs × Credits) / Sum(All Credits)
```

## Common Questions ❓

**Q: Where is my data stored?**  
A: Your browser's local storage. Nowhere else!

**Q: Can I use this offline?**  
A: Yes! After first load, it works completely offline.

**Q: Will I lose my data?**  
A: Only if you clear browser cache. Data persists otherwise.

**Q: How do I add custom subjects?**  
A: Click "Add Subject" in manual entry mode.

**Q: Can OCR detect handwritten marksheets?**  
A: Best results with printed/typed marksheets.

## Troubleshooting 🔧

| Problem | Solution |
|---------|----------|

| Data disappeared | Check browser's LocalStorage settings |
