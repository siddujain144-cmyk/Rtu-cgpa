## 🎓 How It Works

### User Journey

```
1️⃣ Open calculator
2️⃣ Select Branch (CSE, ECE, EE, ME, CE, IT)
3️⃣ Select Semester (1-8)
4️⃣ Choose subjects (with choice options)
5️⃣ Select grades (A++ to F)
6️⃣ Calculate SGPA
7️⃣ Save result
8️⃣ View CGPA across all semesters
9️⃣ Share results!
```

### Behind the Scenes
- Calculates: `SGPA = Σ(Grade × Credits) / Σ(Credits)`
- Converts: `Percentage = (SGPA / 10) × 100`
- Saves: Automatic LocalStorage backup
- Aggregates: Auto CGPA from all semesters
